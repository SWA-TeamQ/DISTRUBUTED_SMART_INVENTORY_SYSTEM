# DSIMS Documentation

This folder contains the documentation set for the Distributed Smart Inventory Management System (DSIMS). The documents explain what will be built, how it is structured, how it communicates over RMI, how data is stored, and how to run and test the system.

## Documentation Index
1. **Overview** – project goals, scope, and key features.  
   File: `01-overview.md`

2. **Requirements** – functional and non-functional requirements.  
   File: `02-requirements.md`

3. **Architecture** – tiered layers, components, and package responsibilities.  
   File: `03-architecture.md`

4. **Setup & Run** – prerequisites and local run steps for server/client.  
   File: `04-setup.md`

5. **RMI API** – remote interface contract and method semantics.  
   File: `05-rmi-api.md`

6. **Data Model & Persistence** – database schema and file structure.  
   File: `06-data-model.md`

7. **Concurrency Model** – locking strategy and UI threading rules.  
   File: `07-concurrency.md`

8. **Logging & Errors** – audit logging and exception handling strategy.  
   File: `08-logging-errors.md`

9. **Testing Plan** – unit, integration, and concurrency tests.  
   File: `09-testing.md`

10. **Roadmap** – sprint plan and deliverables.  
    File: `10-roadmap.md`
